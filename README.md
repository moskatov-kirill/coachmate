# CoachMate

Персональный Telegram-ассистент для тренера на базе ИИ (GigaChat) и Obsidian.

## Стек технологий

*   **Язык:** Python
*   **Фреймворк для API:** FastAPI
*   **База данных:** SQLite
*   **Оркестратор:** OpenClaw
*   **ИИ-модель:** GigaChat
*   **API фитнес-трекеров:** Polar API
